/////////////////////// API Economics-stats ///////////////////////////////

GET a la ruta base
curl -v -XGET -H 'Content-type: application/json'  'https://sos1617-12.herokuapp.com/api/v1/economics'

POST a la ruta base
curl -v -XPOST -H 'Content-type: application/json' -d '{ "province": "Granada", "year": "2006", "expensivepeu": 2544125, "expensiveid": 25336, "employersid": 5412 }' 'https://sos1617-12.herokuapp.com/api/v1/economics'

GET a un recurso 
curl -v -XGET -H 'Content-type: application/json'  'https://sos1617-12.herokuapp.com/api/v1/economics/Sevilla/2009'

DELETE a un recurso
curl -v -XDELETE -H 'Content-type: application/json'  'https://sos1617-12.herokuapp.com/api/v1/economics/Sevilla/2009'

PUT a un recurso
curl -v -XPUT -H 'Content-type: application/json' -d '{ "province": "Sevilla", "year": "2009", "expensivepeu": 999999, "expensiveid": 99999, "employersid": 9999 }' 'https://sos1617-12.herokuapp.com/api/v1/economics/Sevilla/2009'

POST a un recurso  debe dar un error de método no permitido.

/////////////////////////Free Software Stats Test///////////////////////////////
